Select what to show.

After pressing Show Results a window opens with a list of all result files found in your BenchIt directory.
If you can not see anything, you may have not run any BenchIt test jet, or your BenchIt path isn't set correctly [see admin].
You can choose one or more results by using the Ctrl- or Shift-key. Then press Show Selected, if you have the jfreechart and jcommon libraries the results are shown in a fine java created diagram [more], if not, gnuplot ang gv are used.

Working with the java diagram.

You have two possibilities to look at the results: graphical and text only. You can switch between both views by clicking on the tabs plot and text. There is a third tab (config) for configurations.

the result as text
This is a direct look at the result file created by BenchIt. There is a lot of information displayed, which is not completely shown in the graphical view.

the result as graphical plot
Here you see the result plotted as a fine diagram.
You can zoom in by dragging a frame from top left to bottom right, to zoom out drag a frame from bottom right to top left. Also you can zoom by using the PopupMenu that is opening on right click over the diagram.
You have also the possibility to print or save the diagram or to change some properties by using the PopupMenu.
Properties... allows you to change some properties which belong to the plot.
Save as... allows you to save the diagram to disk as a png.
Print... allows you to print the diagram.

the config tab
Here is a button for disabling or reenabling the two blue lines following the pointer (called crosshair).
You may also specify custom ranges for the achses of the plotwindow. By pressing the button set the values are saved and the plot is repainted. The button reset restores the default values.
